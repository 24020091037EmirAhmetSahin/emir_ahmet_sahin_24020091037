# Emira Not Defteri

Firebase Firestore tabanlı gerçek zamanlı not defteri uygulaması.

## Hızlı Başlangıç

```bash
npm install
npx expo start
```

- `w` — Web tarayıcısında aç
- `a` — Android emülatörde aç
- `i` — iOS simülatörde aç (macOS gerekir)

## Geliştirici

**Emira** — Veri Tabanları Ödev 7

## Dosyalar

| Dosya | Açıklama |
|-------|----------|
| `src/app/index.tsx` | Not listesi ve CRUD işlemleri |
| `src/app/hakkimda.tsx` | Proje ve geliştirici bilgileri |
| `src/emiraFirebaseConfig.js` | Firebase yapılandırması |
